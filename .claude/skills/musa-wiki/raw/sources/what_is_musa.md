# 什么是 MUSA

元计算统一系统架构 MUSA (Meta-computing Unified System Architecture)，是摩尔线程推出的 GPU 并行计算平台和编程语言。元计算一词涵盖人工智能计算、图形计算、物理仿真计算等重要领域。

MUSA 是通用并行计算架构，包含 MUSA 指令集（ISA）以及 GPU 内部的并行计算引擎，使 GPU 能够解决复杂的计算问题。

MUSA 软件栈是并行计算编程语言，面向并行计算程序的开发人员，是连接摩尔线程 GPU 与并行计算程序的桥梁。开发人员使用 C/C++ 语言为 MUSA 架构编写程序，程序可在支持 MUSA 的 GPU 上以超高性能运行。同时，MUSA 软件栈还提供各领域的加速库，供开发者直接使用，便于快速适配各种高性能计算场景。
