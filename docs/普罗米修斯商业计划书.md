# 普罗米修斯（Prometheus）——商业计划书

> 基于MTClaw的自我进化型个人认知智能体

---

## 一、项目概述

### 1.1 项目名称

**普罗米修斯（Prometheus）** —— 基于MTClaw的自我进化型个人认知智能体

### 1.2 一句话描述

让AI从"会回答"进化为"会思考、会成长"，打造每一位用户专属的、真正"懂你"的数字伙伴。

### 1.3 项目背景

在信息爆炸的时代，知识散落在文档、邮件、笔记和聊天记录中。现有的知识管理工具存在三大痛点：

- **被动检索**：只能关键词匹配，无法理解用户意图
- **无法持续学习**：每次对话从零开始，不积累经验
- **不会主动服务**：用户不问不答，无法预判需求

普罗米修斯致力于改变这一切。

### 1.4 项目定位

本项目基于**摩尔线程MTClaw开源AI智能体框架**，在**摩尔线程MTT AIBOOK本地运行**，打造一款具备长期记忆与自我进化能力的个人AI智能体。它不再是一个被动问答的知识库，而是一个能持续学习、主动思考、不断进化的"数字伙伴"。

**部署模式**：支持**本地知识库**（数据不出设备）和**云上知识库**（通过 API 访问，无需本地部署）双模式。用户可根据隐私需求灵活选择——敏感数据留在本地，日常知识上云即可用，两者可无缝混合使用。

## 三、核心创新点

### 3.1 多维记忆系统

融合三种记忆类型，模拟人类记忆机制：

| 记忆类型 | 功能 | 技术实现 |
|---------|------|---------|
| **短期工作记忆** | 当前会话上下文，毫秒级响应 | 上下文窗口 + 注意力机制 |
| **长期情节记忆** | 记住用户偏好、习惯、历史交互 | 向量数据库 + 情节编码 |
| **程序性记忆** | "学会"如何完成复杂任务 | 任务模板 + 技能图谱 |

**关键突破**：三种记忆联动运作——情节记忆提供上下文，程序性记忆提供方法，工作记忆执行任务。

### 3.2 反思与进化引擎

引入**"快慢双循环"持续学习机制**：

```
┌─────────────────────────────────────────────────┐
│                   快循环（前台）                    │
│  ┌──────────┐   ┌──────────┐   ┌──────────┐    │
│  │ 用户输入  │ → │ 意图识别  │ → │ 毫秒响应  │    │
│  └──────────┘   └──────────┘   └──────────┘    │
│                                                     │
│                   慢循环（后台）                    │
│  ┌──────────┐   ┌──────────┐   ┌──────────┐    │
│  │ 交互记录  │ → │ 反思总结  │ → │ 策略更新  │    │
│  └──────────┘   └──────────┘   └──────────┘    │
└─────────────────────────────────────────────────┘
```

- **快循环**：高频轻量操作，由"前台助理"毫秒级响应
- **慢循环**：复杂推理转交大模型；后台定期反思过往交互，总结经验、优化策略，实现能力的自主进化

### 3.3 自组织知识库

- 将碎片化信息**自动编译**为带双向链接的知识网络
- 知识随交互**动态生长**，而非静态存储
- 将"检索"升级为"推理"——AI理解知识间的关联，而非简单匹配关键词

### 3.4 主动服务能力

突破"用户不问不答"的被动模式：

- **场景感知**：基于用户当前工作上下文理解需求
- **意图预判**：结合长期记忆预判下一步行动
- **主动推送**：在恰当时机提供关键信息、决策建议

## 四、市场调研

### 4.1 宏观市场：AI Agent 正处爆发前夜

全球 AI Agent 市场正经历指数级增长。多方研究机构给出了一致的方向判断：

| 研究机构 | 2025年规模 | 2030年预测 | CAGR |
|---------|-----------|-----------|------|
| MarketsandMarkets | $52亿 | $471亿 | 44.8% |
| Grand View Research | $54亿 | $1987亿（2033） | 44.2% |
| Bloomberg Intelligence | — | $2500亿+ | — |

**关键判断**：AI Agent 不是一时的风口，而是继搜索引擎、社交媒体、移动应用之后的第四代计算范式。2025-2027 是市场从"早期采用者"跨越到"早期大众"的关键窗口——这正是普罗米修斯的目标时间窗。

### 4.2 细分市场一：个人知识管理（PKM）

| 指标 | 数据 | 来源 |
|------|------|------|
| 全球知识工作者 | ~10 亿人 | ILO / World Bank |
| PKM 软件市场规模（2025） | ~$18 亿 | 综合估算 |
| 预计 2030 规模 | ~$45-60 亿 | CAGR 14-18% |
| PKM 工具渗透率 | <5%（相对知识工作者总量） | 巨大增长空间 |

**头部玩家数据**：
| 产品 | 用户量 | 年收入（估算） | 模式 |
|------|-------|-------------|------|
| Notion | 1 亿+ | ~$25 亿 ARR | Freemium + 团队协作 |

- Obsidian 以 ~15 人团队做到了 ~$1500 万 ARR，证明了"小团队 + 本地优先"模式的商业可行性
- 两者之间（400 万 → 1 亿用户）存在巨大空白——**一个有 AI 原生记忆能力的 PKM 产品**正好填补这个空白

### 5.2 关键技术栈

### 5.3 云上知识库（Prometheus Cloud）

**核心思路**：用户无需部署本地环境，知识库托管于 Prometheus Cloud，通过 API 获取全部智能体能力。

| 特性 | 本地模式 | 云上模式 |
|------|---------|---------|
| 部署门槛 | 需要 MTT AIBOOK | 零部署，注册即用 |
| 知识库存储 | 本地文件系统 | Prometheus Cloud 托管 |
| 推理算力 | 本地 GPU | 云端 GPU 集群 |
| 隐私级别 | 数据不出设备 | 端到端加密 + 租户隔离 |
| 适用场景 | 敏感数据、离线环境 | 日常知识管理、快速上手 |
| 访问方式 | 桌面应用 / 命令行 | Web 浏览器 + API + 桌面应用 |

两种模式共享同一套智能体核心——记忆系统、反思引擎、知识图谱——用户可随时在两种模式间迁移数据。

### 5.4 MVP功能范围

**Phase 1（核心闭环）**：
- 基础对话交互（MTClaw框架集成）
- 短期工作记忆（会话上下文管理）
- 文档索引与检索（本地文件系统 + 云上知识库 API）
- 基础知识图谱（双向链接）
- 云上知识库 MVP（注册即用，零部署）

**Phase 2（记忆与进化）**：
- 长期情节记忆（用户偏好学习）
- 后台反思引擎（慢循环）
- 主动推送通知

### 6.7 收入引擎六：社区生态（Obsidian 插件模式）

> Obsidian 社区 2000+ 插件全部免费，官方不抽成。插件的价值是**拉新和留存**，而非直接变现。普罗米修斯采用相同策略。

| 生态组件 | 收费模式 | 平台角色 |
|---------|---------|---------|
| **社区插件** | 完全免费，官方不抽成 | 官方维护插件市场 + 审核标准 |
| **官方技能包** | 免费，作为获客手段 | 官方开发标杆技能（"律师工作台""学术研究助手"） |
| **知识模板** | 社区免费 + 官方精选免费 | 降低新用户上手门槛 |
| **第三方模型适配** | 社区贡献，官方不收费 | 适配更多国产 GPU / NPU 的社区驱动 |

**变现逻辑**：插件不赚钱，插件驱动的**用户增长和留存**赚钱。一个用户在普罗米修斯中装了 20 个插件、构建了知识图谱、训练了个人记忆——迁移成本极高，留存自然稳固。

### 6.11 获客策略
